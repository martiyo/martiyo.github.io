EWW, el Emacs Web Wowser, es un navegador web para GNU Emacs. Puede cargar, analizar y mostrar varias p√°ginas web usando shr.el. Sin embargo, se requiere un GNU/Emacs con soporte para libxml2.

 Definir eww como navegador por defecto
 #+BEGIN_SRC
 (setq browse-url-browser-function 'eww-browse-url)
 #+END_SRC

 Cambiar directorio de descargas
 #+BEGIN_SRC
 '(eww-download-directory "~/storage/descargas")
 #+END_SRC
